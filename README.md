# 🚨 Sentinel-Stream  
**AI-Driven Continuous Behavioral Authentication & Autonomous Security Response**

Sentinel-Stream is a **frontend prototype** that demonstrates how AI can continuously authenticate users based on behavioral biometrics (keystrokes, mouse dynamics, navigation patterns) and autonomously trigger security actions when anomalies are detected.

This project is designed for **hackathons, demos, and concept validation**, focusing on **UX, explainability, and agentic AI workflows** rather than backend enforcement.

---

## 🧠 Core Idea

Traditional authentication ends at login.

**Sentinel-Stream authenticates continuously.**

It simulates:
- Real-time trust score computation
- Behavioral anomaly detection
- AI agent decision-making
- Autonomous protective actions (block, lock, notify)

---

## ✨ Key Features

### 🔐 Continuous Behavioral Authentication
- Keystroke rhythm analysis
- Mouse movement dynamics
- Session consistency monitoring
- Trust score evolution over time

### 📉 Risk Detection Engine (Prototype)
- Trust score drop simulation
- Normal vs suspicious behavior comparison
- Visual anomaly timelines
- Explainable AI signals

### 🤖 Agentic AI Flow
- Observe → Analyze → Decide → Act pipeline
- Autonomous security responses
- Zero human intervention (simulated)

### ⚡ Autonomous Actions
- Transaction blocking
- Account locking
- Security team notification
- Compliance-ready audit logs

### 📊 Visual Analytics
- Trust score timelines
- Behavioral bar charts
- Risk comparison dashboards
- Action execution traces

## 🛠️ Tech Stack

- **Frontend:** React 18 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS + Glassmorphism UI
- **Animations:** Motion
- **Charts:** Recharts
- **Icons:** Lucide React
- **Routing:** React Router
- **Deployment:** Vercel

> ⚠️ This is a **frontend-only prototype**.  
> All AI models, trust scores, and decisions are **simulated** to demonstrate system behavior and UX.
